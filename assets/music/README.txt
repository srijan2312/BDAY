MUSIC INSTRUCTIONS:

1. Add your special song files to this directory
2. Recommended format: MP3 (best compatibility across browsers)
3. For background music:
   - Name the file: background-music.mp3
   - Choose a gentle, uplifting song that's not too distracting

4. For the music player:
   - Name the file: special-song.mp3
   - This should be "your song" or her favorite song

5. After adding music files, edit the script.js file if you need to change the titles 
   and artist names displayed in the player

NOTE: Music will only play when a user interacts with the page (browser security rule).
